This project is a rewrite of the classic arcade game, R-\/\+TYPE, in C++ with added multiplayer functionality via server-\/client communication.\hypertarget{md_README_autotoc_md106}{}\doxysection{Features}\label{md_README_autotoc_md106}

\begin{DoxyItemize}
\item Multiplayer mode, with support for multiple players to connect and play together
\item Server-\/client communication, allowing players to play together in real-\/time
\item Cross-\/platform compatibility, with support for Windows, Linux
\item Complete Entity Component System (ECS)
\end{DoxyItemize}\hypertarget{md_README_autotoc_md107}{}\doxysection{Requirements}\label{md_README_autotoc_md107}

\begin{DoxyItemize}
\item A C++17 compiler
\item cmake for Linux or Visual Studio 22 for Windows
